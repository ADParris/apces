## Project: AP Clothing E-commerce Site

### Client

The client was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Added [Styled Components](https://www.npmjs.com/package/styled-components) to handle styling.

Added [Open Sans Condensed](https://fonts.google.com/specimen/Open+Sans+Condensed) for site font.

Added [React Router DOM](https://www.npmjs.com/package/react-router-dom) to handle intra-app routing.

Added [Firebase](https://www.npmjs.com/package/firebase) to handle authentication and database.

Added [Redux](https://www.npmjs.com/package/redux) to handle the state of the application.

Added [React Redux](https://www.npmjs.com/package/react-redux) bindings for Redux to work with React.
